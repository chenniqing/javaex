# javaex前端框架

#### 项目介绍
由java程序员以后端开发者的使用角度而设计并开发的前端框架，不修改原生html、js语法，不创造新的语法。

基于原生、忠于原生，零门槛快速上手。

基于jquery开发，不改变任何原生html的写法与取值，只需具备基础的html、js和jquery知识即可。
